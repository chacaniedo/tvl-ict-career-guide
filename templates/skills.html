Essential ICT Skills

🧠 Problem Solving & Logic – Ability to analyze and solve technical issues efficiently.

💻 Programming – Understanding of HTML, CSS, Python, Java, and basic coding logic.

🌐 Networking & Internet Fundamentals – Knowledge of LAN/WAN setup, routers, and IP configuration.

🛠️ Computer Hardware Servicing – Skills in assembling, repairing, and maintaining computer systems.

🗄️ Database Management – Understanding how to use MySQL or Firebase to store and organize data.

🧩 IDE Knowledge – Familiarity with tools like Visual Studio Code, Eclipse, or PyCharm for coding.

⚙️ Software Installation & Troubleshooting – Ability to set up, configure, and fix basic software issues.

📊 Using Productivity Tools – Proficiency in Google Suite, MS Office, and online collaboration tools.

💬 Communication & Teamwork – Collaborating effectively with peers and presenting technical ideas clearly.
